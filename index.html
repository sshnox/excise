What is C2PA?

C2PA (Content Credentials) is a provenance standard that embeds a signed manifest inside an image — recording how it was created and edited, including whether AI was involved. Many camera apps and AI image tools now attach it automatically.

The manifest lives in APP11 / JUMBF segments in JPEGs and caBX chunks in PNGs. excise finds and removes exactly those, leaving the pixels untouched.

How it works

For JPEG and PNG, excise edits the file structure directly and drops only the metadata segments — no re-compression, no quality loss. Other formats fall back to a canvas re-encode, which may recompress.

Because it's all client-side, you can save this page and use it offline. Nothing is logged, stored, or sent.

Use responsibly — stripping provenance is great for removing location and device data before sharing, but respect authenticity where it matters.
